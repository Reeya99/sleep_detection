# Sleep Detection

A drowsiness detection project using deep learning.

## Dataset
Dataset used:  [Drowsiness Dataset] https://www.kaggle.com/datasets/dheerajperumandla/drowsiness-dataset from Kaggle

## How to Run
1. Download the dataset from the Kaggle link above
2. Place the `train` folder in the project directory
3. Install dependencies and run the notebook or `app.py`
